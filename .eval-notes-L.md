```
printf '<3 tools/call frames>' | mcptrace proxy stdio \
    --store jsonl --store-path spans.jsonl --exporter stdout \
    -- node echo_server.js
```

metrics line `spans_emitted=3 dropped_buffer_full=0 dropped_id_collision=0 frames_too_large=0 parse_errors=0 orphans=0`.

- `arg_digest` differs across "search q=hello" vs "search q=world" — request params canonicalised correctly.
- `tool_name` extracted from `params.name` for `tools/call`.
- `req_bytes` / `res_bytes` match wire frames.
- `status="ok"`, `error_code=null`.
- Spawn a real OS child, two real OS pipes, real tokio reactor, real JSONL append. End-to-end works on Windows.
